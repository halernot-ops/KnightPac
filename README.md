# ⚔️ KnightPac

Modern package manager GUI for Arch Linux.

## Features

- Package search
- Package installation
- Package removal
- AUR support (planned)
- Modern PySide6 interface

## Status

Current version: **0.5 Alpha**

Project is under active development.
